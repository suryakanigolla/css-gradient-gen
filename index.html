How to use the CSS Gradient Generator?

  • Gradient Bar on the right can be used to change the percentage influence of each color

  • Clicking on Gradient Bar will add a new pointer with colour value equal to the hex code on bottom

  • You can use the RGB sliders and the colour picker to select a colour

  • Click on Edit Pointers on top left to delete a pointer

  • Finally, Get CSS button gives you the CSS code
